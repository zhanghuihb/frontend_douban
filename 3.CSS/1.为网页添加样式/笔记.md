# 为网页添加样式

## 术语解释

CSS规则 = 选择器 + 声明块

### 选择器

选择器： 选中元素

1.元素选择器
2.ID选择器
3.类选择器 （用的最多）

### 声明块

```
    h1 {
        color: red;
        background-color: lightblue;
        text-align: center;
    }
```
 出现在大括号中

 声明块中包含很多生命，每一个声明表达了某一方面的样式

## 内部样式表

1. 内部样式表

书写在style中

2. 内联样式表

直接书写在元素的style属性中

3. 外部样式表[用的最多，推荐使用]

将样式书写到独立的css文件中

优点：
1)可以解决多页面样式重试的问题
2)有利于浏览器缓存，提高页面响应速度
3)有利于代码分离（HTML和CSS）, 更容易阅读和维护
